# Chapter 2 Mini Project: Profile Card

Build a profile card using only `React.createElement` (no JSX). The card should include:

1. An avatar/photo placeholder (a div with styled initials)
2. A name (h2)
3. A short bio paragraph
4. A list of skills/tags
5. Social media links (3 buttons)

## Requirements

- Use only `React.createElement` (no JSX)
- Use CDN for React (like in `react.html`)
- Style with CSS classes
- Make it look clean and modern

## Bonus

- Add a theme toggle (light/dark) using a button
- Show your card in a centered layout

## Starter structure

```
profile-card/
  index.html
```
